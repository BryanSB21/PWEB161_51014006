Berikut persyaratan Tugas Pertama Praktikum Pemograman WEB:

  1. Cari minimal 5 lagu anak
  2. Cari gambar untuk setiap lagu
  3. Buat halaman index.html berisi daftar lagu
  4. Buat halaman html untuk masing-masing lagu, berisi syair dan gambar untuk lagu
  5. Desain halaman secantik mungkin (tanpa CSS)
  6. Ada link kembali ke halaman index dari setiap lagu
  7. Judul lagu di link ke wikipedia bahasa Indonesia untuk istilah terkait
  8. Dari halaman index ada link ke halaman Facebook Anda
  9. Semua file disimpan dalam folder bernama "Tugas #1" dalam GitHub Anda
